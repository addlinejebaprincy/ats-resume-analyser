# ResumeAI

ResumeAI is an AI-powered ATS (Applicant Tracking System) Resume Analyzer that helps users evaluate their resumes for ATS compatibility, compare them with job descriptions, and receive AI-powered suggestions to improve their chances of getting interviews.

---

## 🚀 Features

- Modern and responsive SaaS landing page
- ATS resume score analysis
- AI-powered resume improvement suggestions
- Job description matching
- Keyword analysis
- Resume upload functionality
- Interactive analytics dashboard

---
